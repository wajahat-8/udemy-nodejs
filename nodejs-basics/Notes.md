1. HTTP (http)

Used to create servers and handle HTTP requests and responses.

const http = require("http");

2. HTTPS (https)

Used to create secure servers or make HTTPS requests.

const https = require("https");

3. File System (fs)

Provides methods to work with files and directories.

const fs = require("fs");

4. Path (path)

Helps work with file and directory paths.

const path = require("path");

5. Operating System (os)

Provides system-level information such as CPU, memory, and network details.

const os = require("os");
